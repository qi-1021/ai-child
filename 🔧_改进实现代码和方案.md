## 改进 #1: 移除知识全注问题 ⭐⭐⭐ (10分钟)

### 问题回顾

当前代码在 `server/ai/child.py:184-188`:

```python
all_knowledge = await get_all_knowledge(session)
relevant = await search_knowledge(session, user_text[:64])
seen_ids = {k.id for k in relevant}
for k in all_knowledge:
    if k.id not in seen_ids:
        relevant.append(k)
```

**问题**: 最终 `relevant == all_knowledge`，所有知识都被注入

---

### 改进方案 #1a: 直接移除（最简单）

**文件**: `server/ai/child.py`

**改动**:

```python
# 原代码 (184-188)
all_knowledge = await get_all_knowledge(session)
relevant = await search_knowledge(session, user_text[:64])
seen_ids = {k.id for k in relevant}
for k in all_knowledge:
    if k.id not in seen_ids:
        relevant.append(k)

# 改为
all_knowledge = await get_all_knowledge(session)
relevant = await search_knowledge(session, user_text[:64])

# 限制在 Top-15
relevant = sorted(relevant, 
                 key=lambda k: (k.confidence, -datetime.now().timestamp() + k.timestamp.timestamp()),
                 reverse=True)[:15]
```

**优点**:
- ✅ 直接修复问题
- ✅ 只需移除 3 行
- ✅ 性能提升

**缺点**:
- ⚠️ 如果搜索结果少于 5 条，知识块会很小

### 改进方案 #1b: 混合策略（推荐）

**文件**: `server/ai/child.py`

**改动**:

```python
# 在 chat() 函数中 (替换 184-188)
# 获取相关知识
all_knowledge = await get_all_knowledge(session)
relevant = await search_knowledge(session, user_text[:64])

# 按置信度和相关性排序相关知识
relevant = sorted(relevant, 
                 key=lambda k: k.confidence,
                 reverse=True)[:10]

# 获取背景知识 (始终有效的高置信度知识)
background_knowledge = [
    k for k in all_knowledge 
    if k.confidence == 100 and k.id not in {k.id for k in relevant}
][:5]

# 合并
relevant.extend(background_knowledge)
```

**效果**:
- ✅ 最多 15 条知识 (10相关 + 5背景)
- ✅ 平衡相关性和背景覆盖
- ✅ Token 成本可预测

**示例流程**:
```
用户: "怎样治疗感冒?"

  5. 警示信号 (confidence: 75)

背景知识 (confidence=100的其他):
  1. 医学基础知识

  = 10 条相关知识 + 5 条背景知识
```

### 改进方案 #1c: 高级版本（向量相似度优化）

**需要**: 稍后实现，需要嵌入模型

```python
# 伪代码 (需要实现向量相似度)
from sentence_transformers import util

# 获取用户输入的嵌入
user_embedding = embedding_model.encode(user_text, convert_to_tensor=True)

# 计算所有知识的相似度
all_knowledge = await get_all_knowledge(session)
knowledge_embeddings = [embedding_model.encode(f"{k.topic} {k.content}", convert_to_tensor=True) 
                        for k in all_knowledge]

similarities = util.pytorch_cos_sim(user_embedding, knowledge_embeddings)[0]

# 排序并取 Top-15
top_indices = similarities.topk(k=min(15, len(all_knowledge)))[1]
relevant = [all_knowledge[i] for i in top_indices]
```

**优点**: 精准度 95%+  
**缺点**: 需要额外模型，成本更高

## 改进 #2: 置信度筛选 ⭐⭐⭐ (5分钟)

### 问题

所有知识都被注入，即使是低置信度的知识。

### 解决方案

**文件**: `server/ai/child.py`

**修改 `_build_context()` 函数**:

```python
def _build_context(
    history: List[Conversation],
    knowledge_items: List,
    name: str | None,
    is_sleeping: bool = False,
) -> List[Dict[str, Any]]:
    """Build the OpenAI messages list from DB history and knowledge."""
    messages: List[Dict[str, Any]] = [
        {"role": "system", "content": _build_system_prompt(name, is_sleeping)}
    ]

    if knowledge_items:
        # 📍 新增: 按置信度过滤
        # 默认只注入 confidence >= 70 的知识
        high_quality = [k for k in knowledge_items if k.confidence >= 70]
        
        # 如果没有高质知识，使用全部 (降级方案)
        if not high_quality:
            high_quality = knowledge_items
            logger.warning("No high-confidence knowledge items found, using all items")
        
        # 格式化知识
        kb_text = "\n".join(
            f"- [{item.topic}] {item.content}" for item in high_quality
        )
        messages.append(
            {
                "role": "system",
                "content": f"Things you have learned so far:\n{kb_text}",
            }
        )

    for msg in history:
        messages.append({"role": msg.role, "content": msg.content})

    return messages
```

**可选优化**: 显示置信度指示器

```python
# 在知识格式中加入置信度标记
def _format_knowledge_items(items: List, min_confidence: int = 70) -> str:
    """Format knowledge items with confidence indicators."""
    high_quality = [k for k in items if k.confidence >= min_confidence]
    if not high_quality and items:
        high_quality = items
    
    lines = []
    for item in high_quality:
        # 使用符号表示置信度
        if item.confidence >= 90:
            badge = "🟢"  # 高置信
        elif item.confidence >= 70:
            badge = "🟡"  # 中等置信
        else:
            badge = "🔴"  # 低置信
        
        lines.append(f"- {badge} [{item.topic}] {item.content}")
    
    return "\n".join(lines)

# 在 _build_context() 中使用
kb_text = _format_knowledge_items(knowledge_items)
```

## 改进 #3: 集成性格系统 ⭐⭐⭐ (15分钟)

### 问题

### 解决方案

**文件**: `server/ai/child.py`

**第一步**: 导入性格相关函数

**第二步**: 修改 `_build_context()` 函数

```python
async def _build_context(
    session: AsyncSession,  # 📍 新增参数
    history: List[Conversation],
    knowledge_items: List,
    name: str | None,
    is_sleeping: bool = False,
) -> List[Dict[str, Any]]:
    """Build the OpenAI messages list from DB history and knowledge."""
    messages: List[Dict[str, Any]] = [
        {"role": "system", "content": _build_system_prompt(name, is_sleeping)}
    ]

    # 📍 新增: 获取性格背景
    try:
        manager = PersonalityMemoryManager(session)
        personality_context = await manager.build_personality_context()
        if personality_context:
            messages.append({
                "role": "system",
                "content": personality_context
            })
    except Exception as e:
        logger.warning(f"Failed to load personality context: {e}")

    # 原有的知识块逻辑
    if knowledge_items:
        high_quality = [k for k in knowledge_items if k.confidence >= 70]
        if not high_quality:
            high_quality = knowledge_items
        
        kb_text = "\n".join(
            f"- [{item.topic}] {item.content}" for item in high_quality
        )
        messages.append(
            {
                "role": "system",
                "content": f"Things you have learned so far:\n{kb_text}",
            }
        )

    # 对话历史
    for msg in history:
        messages.append({"role": msg.role, "content": msg.content})

    return messages
```

**第三步**: 更新 `chat()` 函数调用

```python
# 原来 (line 189)
messages = _build_context(history, relevant, name, is_sleeping)

# 改为
messages = await _build_context(session, history, relevant, name, is_sleeping)
```
