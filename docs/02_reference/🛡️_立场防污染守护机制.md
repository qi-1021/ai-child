**版本**: 1.0

## 📌 核心原则

### 问题陈述

AI Child 需要学习教授者（用户）的观点和立场，但互联网充满了：
- ❌ 过时的信息
- ❌ 偏见和不实陈述
- ❌ 低质量内容
- ❌ 被污染的数据

**关键问题**：如何让 AI 学习用户的立场，同时防止被网络污染？

---

## 🎯 解决方案：源权重系统

### 核心设计

我们在 PersonalityMemory 表中引入**三个关键机制**：

```
来源权重 (source_authority)
    ↓
信息质量评级
    ↓
立场相关性 (is_stance_relevant)
    ↓
是否影响 AI 的个性
```

---

## 🔑 三个关键字段

### 1️⃣ 来源权重 (source_authority)

**权重系统**：

| 来源 | 权重 | 说明 |
|------|------|------|
| `user` | **100** | 教授者的观点 ⭐ 最高权威 |
| `self` | **80** | AI 自己的推理和洞察 |
| `system` | **75** | 系统提供的默认值 |
| `inference` | **50** | AI 的推断（需要验证） |
| `web` | **40** | 网络数据 ⚠️ 容易污染 |
| `verified_web` | **70** | 验证过的网络信息 |

**使用规则**：
```python
# 添加来自用户的观点
await manager.add_memory(
    source="user",              # 自动设置 authority=100
    content="我相信诚实最重要",
    is_stance_relevant=True     # 立即影响 AI 的个性
)

# 添加网络信息
await manager.add_memory(
    source="web",               # 自动设置 authority=40
    content="2026年新的编程语言是...",
    is_stance_relevant=False    # ← 自动降级：不影响立场
)
```

### 2️⃣ 立场相关性 (is_stance_relevant)

**定义**：这条记忆是否会影响 AI 的**个性、价值观、观点**？

| 值 | 含义 | 示例 |
|----|------|------|
| `True` | 影响 AI 的核心个性 | "我喜欢提问"、"我相信诚实" |
| `False` | 仅作为知识/信息 | 技术事实、历史信息 |

**关键规则**：
```
用户声明              → is_stance_relevant=True （无条件相信）
用户反馈/教学         → is_stance_relevant=True（学习用户的观点）
网络来源              → is_stance_relevant=False（污染防护）
验证过的信息          → is_stance_relevant=True（可信）
```

### 3️⃣ 验证状态 (verified_at)

**机制**：
```
web 数据 (authority=40)
    ↓
用户或系统验证
    ↓
verified_at = timestamp
source_authority = 70  (升级)
is_stance_relevant = True (可能)
```

## 🧠 建立 AI 的立场体系

### 三个来源层级

```
┌─────────────────────────────────────────┐
│ 第1层：教授者立场 (source=user)         │
│ authority=100, is_stance_relevant=True  │
│ "我相信好的代码..."                     │
│ "我鼓励持续学习..."                     │
│ ← AI 直接采纳为身份                     │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│ 第2层：经验观点 (source=self)           │
│ authority=80, is_stance_relevant=True   │
│ "我通过交互学到..."                     │
│ "我观察到的模式是..."                   │
│ ← AI 的自主发展                         │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│ 第3层：外部信息 (source=web)            │
│ authority=40, is_stance_relevant=False  │
│ "互联网上的技术信息..."                 │
│ "社交媒体上的意见..."                   │
│ ← 参考数据，不影响个性                  │
└─────────────────────────────────────────┘
```

### 生成 AI 立场的算法

```python
async def build_ai_stance_profile():
    """构建 AI 的立场档案（只包含有权重的观点）"""
    
    # 只获取真正影响立场的记忆
    memories = await manager.get_stance_defining_memories(limit=20)
    
    stance = {}
    for mem in memories:
        if mem.source == "user":            # 第一优先级
            stance[mem.category] = mem.content
        elif mem.source == "verified_web":  # 用户验证的
            if mem.category not in stance:
                stance[mem.category] = mem.content
        elif mem.source == "self":          # AI 自己的观点
            if mem.category not in stance:
                stance[mem.category] = mem.content
    
    return stance  # 教授者的观点优先！
```

```python
# 找出所有可能的污染源
async def find_web_pollution():
    result = await session.execute(
        select(PersonalityMemory)
        .where(
            (PersonalityMemory.source == "web")
            & (PersonalityMemory.is_stance_relevant == False)
        )
    )
    pollutants = result.scalars().all()
    print(f"发现 {len(pollutants)} 条网络数据（已隔离）")
```

## 🛡️ 污染防护的三重防御

### 防线 1：自动降级

```python
# 任何 web 来源自动被标记为 is_stance_relevant=False
if source == "web":
    is_stance_relevant = False  # ← 污染防护！
```

### 防线 2：权重差异

```
用户的观点: authority=100
网络信息:  authority=40

# AI 在实际推理中将优先权重用户
```

### 防线 3：显式标记

```python
tags=["web:caution"]  # 明确标记可能有风险
```
