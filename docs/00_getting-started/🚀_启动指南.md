🚀 AI Child 一键启动指南

==============================================================================

## ⚡ 最快启动（3步，5分钟）

### 步骤1：获取 OpenAI API Key
1. 访问 https://platform.openai.com/api-keys
2. 创建新的 API Key（或使用现有的）
3. 复制这个 Key

### 步骤2：配置环境
在这个目录中创建 server/.env 文件：

```
OPENAI_API_KEY=sk-your-api-key-here
```

### 步骤3：启动服务器

方案 A - 自动脚本（推荐）
```bash
bash quick_start.sh
cd server
source .venv/bin/activate
python main.py
```

方案 B - 手动启动
```bash
cd server
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## 🧪 测试 API - 进行首次对话

### 使用 API 文档界面（最简单）
1. 打开 http://localhost:8000/docs
2. 找到 "POST /chat/text" 端点
3. 点击 "Try it out"
4. 输入：
   ```json
   {
     "text": "你好，我叫张三"
   }
   ```
5. 点击 "Execute"

### 或使用 curl 命令
```bash
curl -X POST "http://localhost:8000/chat/text" \
  -H "Content-Type: application/json" \
  -d '{"text": "你好，我叫张三"}' \
  | python -m json.tool
```

## 📝 常见问题

### Q1: ModuleNotFoundError: No module named 'fastapi'
A: 虚拟环境未激活或依赖未安装
   ```bash
   source server/.venv/bin/activate
   pip install -r server/requirements.txt
   ```

### Q2: OPENAI_API_KEY not set
A: 缺少 .env 配置
   ```bash
   # 在 server/ 目录创建 .env
   echo "OPENAI_API_KEY=sk-your-key" > server/.env
   ```

### Q3: Connection refused
A: 端口 8000 被占用
   ```bash
   # 使用其他端口
   python main.py --port 8001
   ```

### Q4: Database is locked
A: 旧进程未完全关闭
   ```bash
   # 删除旧数据库文件
   rm -f server/ai_child.db server/ai_child.db-*
   ```

### Q5: 在中国无法连接 OpenAI
A: 需要代理或使用本地 LLM（Ollama）
   参考：AI_CHILD_改进路线图.md 的 Phase 4B

## 🔧 详细配置

### 需要的环境变量（在 server/.env 中）

**必需：**
```
OPENAI_API_KEY=sk-...         # OpenAI API key（从 https://platform.openai.com/api-keys 获取）
```

**可选（有默认值）：**
```
# 数据库
DATABASE_URL=sqlite+aiosqlite:///./ai_child.db

# 服务器
HOST=0.0.0.0
PORT=8000

# LLM 模型
OPENAI_MODEL=gpt-4o
OPENAI_VISION_MODEL=gpt-4o
OPENAI_WHISPER_MODEL=whisper-1

# 记忆系统
MEMORY_CONTEXT_TURNS=20              # 内存中保留多少轮对话
PROACTIVE_QUESTION_INTERVAL=2        # 每隔多少轮主动提问

# 自主研究
RESEARCH_ENABLED=true               # 启用 Web 搜索
RESEARCH_QUERY_COUNT=3              # 每次研究生成多少个搜索查询
RESEARCH_MAX_RESULTS=4              # 每个查询最多获取多少个结果

# 睡眠循环
SLEEP_ENABLED=true                  # 启用睡眠巩固机制
```

## 📊 系统要求

- **Python**: 3.10+（推荐 3.11+）
- **操作系统**: macOS / Linux / Windows / Docker
- **依赖**:
  - FastAPI 0.115.6
  - SQLAlchemy 2.0.36
  - OpenAI Python Client 1.59.3
  - 及其他（自动安装）
- **存储**: ~50MB (数据库增长缓慢)
- **内存**: ~200-500MB
- **网络**: 需要访问 OpenAI API + DuckDuckGo（用于搜索）

## 🎯 启动后可以做的事


监控 AI 的学习过程和思考
