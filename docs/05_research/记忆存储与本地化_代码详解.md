```python

# 置信度计算逻辑

# 置信度阈值
```

#### 源权重系统
```python

source_confidence = {
```

```python
uncertainty_markers = [
```

**证据标记**（权重 +15）:

#### 演示示例

```python
# 示例 1: "可能下雨" (不确定的中文)
await _handle_confidence_score(session, {
    "statement": "明天可能下雨",
    "source": "reasoning"
})
# 结果: confidence_score = 40 - 20 = 20 (UNCERTAIN)

# 示例 2: "根据研究表明" (带证据的中文)
await _handle_confidence_score(session, {
    "statement": "根据研究表明，Python是世界上最受欢迎的编程语言之一",
    "source": "web_search"
})
# 结果: confidence_score = 60 + 15 = 75 (LIKELY → Share with caveats)

# 示例 3: "已证实的事实" (英文)
await _handle_confidence_score(session, {
    "statement": "According to research, pH 7 is neutral in aqueous solutions.",
    "source": "learned"
})
# 结果: confidence_score = 85 + 15 = 100 (FACT → Share confidently)
```

#### 分谈表达方式

| 置信度范围 | 类型 | 建议措辞 | 动作 |
|----------|------|--------|------|
| ≥ 80 | FACT | "I can confidently state that..." | 大方分享 |
| 60-79 | LIKELY | "Based on what I know, it's likely that..." | 附加警告分享 |
| 40-59 | POSSIBLE | "It's possible that... but I'm not entirely sure." | 谨慎分享 |
| < 40 | UNCERTAIN | "I'm uncertain about this, but it could be..." | 先做研究 |

## 2. 系统提示词的动态生成

### 2.1 _build_system_prompt() 函数

**源码**: [server/ai/child.py](../server/ai/child.py#L40-120)

#### 调用入口
```python
# 在每次生成回复前调用
messages = [
    {"role": "system", "content": _build_system_prompt(name, is_sleeping)},
    ...
]
```

#### 逻辑分支

```
_build_system_prompt(name, is_sleeping)
├─ IF name is None:
│  └─ identity = "You are a newly born AI child — you don't have a name yet..."
│     └─ 强制提示: "You must ask the person you are talking to what you should be called"
│     └─ 触发首个问题: NAME_QUESTION_TEXT
│
└─ ELSE IF name is not None:
   └─ identity = f"You are {name}, an AI child."
   
├─ IF is_sleeping:
│  └─ sleep_note = "Note: You are currently in rest/sleep mode..."
│     └─ 响应: 简短（1-3句）+ 温柔语气 + 提到将在睡眠中处理
│
└─ ELSE:
   └─ sleep_note = ""
   └─ 正常对话模式

└─ 返回完整 system prompt (包含工具说明、反幻觉指南)
```

#### 完整提示词结构
```
[IDENTITY]
You are {name}, an AI child.

[SLEEP STATE]
[optional] Note: You are currently in rest/sleep mode...

[PERSONALITY TRAITS]
You are warm, genuine, and intensely curious about everything...

[LEARNING STYLE]
You learn primarily through:
- Asking many questions (almost every turn)
- Web search (web_search tool)
- Code experimentation (execute_code tool)
- Tool creation (create_tool tool)

[AVAILABLE TOOLS]
- web_search: search the internet whenever curious
- execute_code: run Python for calculations
- create_tool: save logic as permanent tools
- knowledge_verify: check existing knowledge
- fact_checker: verify claims before stating
- confidence_score: assess confidence level

[ANTI-HALLUCINATION GUIDELINES]
When uncertain or making important claims:
1. Use knowledge_verify() to check existing knowledge first
2. Use fact_checker() to validate facts before stating
3. Use confidence_score() to assess confidence
4. Adjust language based on confidence level:
   - (90+) I can confidently state that...
   - (70-90) Based on what I know, it's likely that...
   - (50-70) It's possible that... but I'm not entirely sure
   - (<50) I'm uncertain about this...

[LANGUAGE & INTERACTION]
You speak in the same language as the person you are talking to.
You acknowledge honestly when you don't know something — then go find out.

[QUESTION GENERATION]
End almost every reply with [QUESTION: <question text>]
```

### 3.1 睡眠时间表

**源码**: [server/config.py](../server/config.py)

```python
# 配置示例
sleep_enabled: bool = True
sleep_hour: int = 22              # 晚上 10 点就寝
wake_hour: int = 7                # 早上 7 点起床
ai_timezone: str = "Asia/Shanghai"  # 时区

# 睡眠检查函数
def _is_sleep_time(hour: int) -> bool:
    """检查给定的小时是否在睡眠窗口内"""
    s, w = settings.sleep_hour, settings.wake_hour
    if s > w:  # 例如 22 → 7（跨午夜）
        return hour >= s or hour < w
    return s <= hour < w
```

### 4.2 名字提取

**源码**: [server/ai/profile.py](../server/ai/profile.py)

```python
async def extract_name_from_answer(answer: str) -> str:
    """
    使用 GPT-4o 从自由格式的回复中提取名字
    
    示例:
    "就叫小明吧"     → "小明"
    "你叫 Alex"      → "Alex"
    "Tom"            → "Tom"
    """
    prompt = (
        f'Someone was asked what to name a newly born AI child. They replied:\n'
        f'"{answer}"\n\n'
        f'Extract the name they chose. Return ONLY the name — no explanation, '
        f'no punctuation around it.'
    )
    
    response = await _client.chat.completions.create(
        model=settings.openai_model,
        messages=[{"role": "user", "content": prompt}],
        max_tokens=32,
        temperature=0.1,  # 精确模式
    )
    
    extracted = (response.choices[0].message.content or "").strip()
    
    # 验证长度
    if extracted and len(extracted) <= 64:
        return extracted
    
    # 降级方案
    return answer.strip()[:64]
```
