## ❓ 常见疑问

**Q: 这会不会让 AI 变得烦人？**  
A: 不会。有冷却机制和参数控制。调整 `CURIOSITY_QUESTION_COOLDOWN` 就能控制频率。

**Q: 代码改动会很复杂吗？**  
A: 不。精简版只需 50 行新代码 + 3 个新字段。

**Q: 能否随时关闭？**  
A: 能。设置 `CURIOSITY_SYSTEM_ENABLED = False` 一秒关闭。

**Q: 这是否会增加 API 成本？**  
A: 知识缺陷检测会增加 1 个 GPT-4o 调用/回复。成本可控。

**Q: 睡眠机制是否会受影响？**  
A: 不会。睡眠时好奇度不积累，提问冷却。可以调整。
