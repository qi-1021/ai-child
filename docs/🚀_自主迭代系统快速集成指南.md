# 🚀 自主迭代系统 — 快速集成指南

## 5 分钟快速理解

你的 AI 将获得：

| 能力 | 触发条件 | 效果 |
|------|---------|------|
| 🎲 **好奇度提问** | 无条件，基于内部积累 | "我最近在想…" |
| 🔍 **知识缺陷提问** | 发现回答中有不懂的概念 | "这个概念我不太明白…" |
| 📚 **知识深化提问** | 用户教了新知识 | "能不能告诉我更多…" |
| 🧠 **性格学习** | 每次对话后，后台运行 | 记住用户偏好，调整风格 |
| 📊 **质量反思** | 每天总结一次 | 反思哪些问题效果好 |

# 在方法开头（生成回复前）
from ai.curiosity_engine import update_curiosity, try_ask_question

async def answer(self, message: str, ...) -> Tuple[str, Optional[str]]:
    async with get_session() as session:
        # ━━━ 新增代码 ━━━
        await update_curiosity(session)
        curiosity_question = await try_ask_question(session, messages, self.name)
        # ━━━ 结束新增 ━━━
        
        # ... 现有的 AI 回复生成代码 ...
        reply_text = await self._call_gpt4o(messages, system_prompt)
        
        # ... 现有的嵌入式问题提取代码 ...
        embedded_question = extract_embedded_question(reply_text)
        
        # ━━━ 修改：问题优先级 ━━━
        final_question = curiosity_question or embedded_question
        # ━━━ 结束修改 ━━━
        
        if final_question:
            await add_pending_question(session, final_question)
        
        return reply_text, final_question
```

## 故障排除

### 现象：好奇度不增长
```
Debug: 
1. 检查 curiosity_score 字段是否存在
   SELECT curiosity_score FROM ai_profile LIMIT 1;
2. 检查 update_curiosity() 是否被调用
   在 child.py 中添加 print() 语句
3. 检查是否有 session.commit()
```

### 现象：提问过于频繁
```
解决：
1. 增加 CURIOSITY_QUESTION_COOLDOWN（默认 3 分钟）
2. 降低 CURIOSITY_MAX_SCORE（默认 100）
3. 检查是否还有其他提问来源同时触发
```

### 现象：知识缺陷检测不工作
```

   打印 concepts 列表
3. 检查知识库查询是否工作
   直接运行 SQL 查询

### 自定义好奇度增长函数

```python
# 替换 curiosity_engine.py 中的 update_curiosity()

async def update_curiosity_custom(session: AsyncSession) -> float:
    profile = await get_ai_profile(session)
    
    # 示例：只在特定时段增长
    current_hour = datetime.now().hour
    if 9 <= current_hour <= 18:  # 工作时间
        increment = 1.5
    elif 20 <= current_hour <= 23:  # 晚间学习时间
        increment = 2.0
    else:
        increment = 0.5  # 深夜睡眠时间
    
    profile.curiosity_score = min(100, profile.curiosity_score + increment)
    await session.commit()
    return profile.curiosity_score
```
