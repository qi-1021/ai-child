### ✅ 修复2️⃣：置信度过滤（Token -5-10%）

**位置**: `server/ai/memory.py` 新增函数

**新函数**：
```python
async def get_high_quality_knowledge(
    session: AsyncSession, 
    min_confidence: int = 70,  # 仅返回置信度 >= 70
    limit: int = 10             # 最多 10 个
) -> List[KnowledgeItem]:
    """仅返回高质量知识"""
    result = await session.execute(
        select(KnowledgeItem)
        .where(KnowledgeItem.confidence >= min_confidence)
        .order_by(KnowledgeItem.confidence.desc())
        .limit(limit)
    )
    return result.scalars().all()
```

**好处**：
- 低质知识自动过滤
- 回答准确度提升
- 避免用错误信息回答
