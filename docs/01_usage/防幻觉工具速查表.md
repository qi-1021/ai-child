# 防幻觉工具快速参考

## 三个新工具一览

| 工具 | 场景 | 输入 | 输出 |
|------|------|------|------|
| `knowledge_verify` | 检查已学知识 | 话题、关键词 | 已知内容列表 |
| `fact_checker` | 验证声明真伪 | 声明、原因 | 置信度分数、来源 |
| `confidence_score` | 评估确定性 | 声明、来源 | 0-100分、演示风格 |

---

## 工具调用示例

### 1️⃣ knowledge_verify - 验证已学知识

**何时使用:**
- "我对Python装饰器了解多少？"
- 避免重复学习
- 确认有足够的知识基础

**API 调用:**
```json
{
  "tool_name": "knowledge_verify",
  "args": {
    "topic": "Python decorators",
    "keywords": "function, wrapper, pattern"
  }
}
```

**返回示例:**
```json
{
  "found": true,
  "topic": "Python decorators",
  "count": 3,
  "knowledge_items": [
    {
      "id": 42,
      "content_preview": "A decorator is a function that...",
      "created_at": "2024-01-15T10:30:00+00:00"
    }
  ],
  "recommendation": "I already know 3 things about 'Python decorators'."
}
```

---

### 2️⃣ fact_checker - 验证声明

**何时使用:**
- 在声称事实之前
- 来自web_search的结果
- 需要高置信度的声明

**API 调用:**
```json
{
  "tool_name": "fact_checker",
  "args": {
    "claim": "The Eiffel Tower is in Paris",
    "reason": "about to teach the user"
  }
}
```

**返回示例:**
```json
{
  "claim": "The Eiffel Tower is in Paris",
  "verified": true,
  "confidence_score": 95,
  "sources": {
    "learned_knowledge": 1,
    "web_search_results": 5
  },
  "recommendation": "This claim is well-supported by my knowledge."
}
```

**置信度分数解释:**
- **75-100:** ✅ 很好支持，可自信分享
- **50-75:** ⚠️ 部分验证，加上注意事项
- **<50:** ❌ 需要更多验证，不要分享

---

### 3️⃣ confidence_score - 评估置信度

**何时使用:**
- 在进行任何声称之前
- 确定回答措辞
- 向用户解释确定性

**API 调用:**
```json
{
  "tool_name": "confidence_score",
  "args": {
    "statement": "According to research, climate change is accelerating",
    "source": "web_search"
  }
}
```

**返回示例:**
```json
{
  "statement": "According to research, climate change is accelerating",
  "source": "web_search",
  "confidence_score": 78,
  "presentation_style": "FACT",
  "suggested_phrasing": "I can confidently state that...",
  "recommendation": "Share this confidently."
}
```

**演示风格:**
- 🔵 **FACT** (≥80): I can confidently state that...
- 🟡 **LIKELY** (≥60): Based on what I know, it's likely that...
- 🟠 **POSSIBLE** (≥40): It's possible that... but I'm not entirely sure.
- 🔴 **UNCERTAIN** (<40): I'm not confident about this, but it could be...

## 工作流程示例

### 场景 1: 学生提问

```
Student: "What is recursion in programming?"

AI思考过程:
1. knowledge_verify("recursion") 
   → 找到之前学的关于recursion的内容
2. confidence_score("Recursion is when a function calls itself", source="learned")
   → 得分: 92, 风格: FACT
3. 回答: "I can confidently state that recursion is when a function calls itself..."
```

---

### 场景 2: Web搜索结果验证

```
AI通过web_search找到: "Quantum computing was invented in 1982"

AI思考过程:
1. fact_checker("Quantum computing was invented in 1982", reason="from web search")
   → 置信度: 35, 需要验证
2. confidence_score("Quantum computing was invented in 1982", source="web_search")
   → 得分: 45, 风格: POSSIBLE
3. 回答: "It's possible that quantum computing was first explored in 1982, 
   but I'm not entirely sure about the exact date..."
```

---

### 场景 3: 推理和推理

```
User: "If the Earth stopped rotating, what would happen?"

AI思考过程:
1. knowledge_verify("Earth rotation effects")
   → 知识有限
2. confidence_score("Without Earth's rotation, gravity would behave differently", 
   source="reasoning")
   → 得分: 35, 风格: UNCERTAIN
3. 回答: "I'm not confident about this, but it could be that without 
   Earth's rotation, our experience of gravity would change significantly..."
```

## 常见问题

### Q: 三个工具如何一起工作？
A: 通常的流程是：
1. **knowledge_verify** → 检查是否已知
2. **fact_checker** → 验证声明（如需要）
3. **confidence_score** → 确定措辞

### Q: 如果knowledge_verify返回空怎么办？
A: 这意味着你对这个话题没有学到任何东西。建议：
- 使用 web_search 研究这个话题
- 请求用户教你
- 明确表说自己缺乏这方面的知识

### Q: 置信度分数何时低于50？
A: 在以下情况下：
- 来自推理或推论的信息
- 包含很多不确定性标记的声明（maybe, might, possibly）
- 知识不足的话题

### Q: 我应该总是调用所有三个工具吗？
A: 不需要，按需使用：
- 快速问题：只用 confidence_score
- 已知话题：只用 knowledge_verify
- 事实声明：只用 fact_checker
- 复杂情况：组合使用

## 故障排查

### 工具返回错误信息
```json
{"error": "Knowledge verification failed: KeyError"}
```
**解决方案:** 检查数据库连接和 KnowledgeItem 表

### 置信度分数总是很低
**解决方案:** 检查：
- 是否使用了正确的 source 参数
- 声明中是否有太多不确定性标记
- source 参数是否是: learned|web_search|reasoning|inference

### fact_checker 不工作
**解决方案:** 
- 检查 web_search 工具是否正常工作
- 验证数据库中是否有KnowledgeItem
- 查看error字段获取详细信息
