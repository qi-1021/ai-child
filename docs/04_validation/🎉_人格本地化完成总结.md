## 🎯 三大核心功能详解

### 1️⃣ 人格记忆隔离

**问题**: 所有记忆一视同仁，AI 的定义性特征容易被清理

**解决方案**:
- ✨ 新表 `personality_memories` - 永不删除
- 📚 主表 `conversations` - 可清理（30-90天）
- 🔒 严格分类：traits(特征) / values(价值观) / origin(来源) / relationships(关系) / beliefs(信念)

**关键特性**:
- 人格记忆**永不自动删除** ✅
- 可设置**重要程度**(1-100)自动注入系统提示词 ✅
- 支持**有立场(personality)** vs **中立(fact)** 区分 ✅
- 完整的 CRUD API 和管理函数 ✅

**示例使用**:
```python
await manager.add_memory(
    category="values",
    content="我相信诚实胜过效率",
    significance=90,
    is_neutral_fact=False,  # 有立场的人格观
)
```

---

### 2️⃣ 完整的本地化系统（i18n）

**问题**: 系统提示词、消息等硬编码英文，不支持中文

**解决方案**:
- 🌍 i18n 管理器 - 灵活的翻译系统
- 📖 集中的消息字典 - 所有字符串统一管理
- 🎯 完整的 en-US 和 zh-CN 覆盖 (100%)

**已翻译的组件**:
```
✅ 问候 (greeting)
✅ 名字相关 (name_question, name_accepted)
✅ 睡眠消息 (sleep.message.{sleep,wake})
✅ 好奇心提问 (question.*)
✅ 知识学习 (knowledge.*)
✅ 工具创建 (tool.*)
✅ 错误处理 (error.*)
✅ 系统提示词 (SYSTEM_PROMPTS)
```

**三种使用方式**:
```python
# 方式 A：i18n 实例
i18n = get_i18n("zh-CN")
msg = i18n.t("sleep.message.wake")

# 方式 B：快速翻译
msg = t("greeting", language="en-US")

# 方式 C：系统提示词
prompt = i18n.system_prompt(ai_name="小明", is_sleeping=False)
```

**支持的语言**:
| 代码 | 语言 | 覆盖率 |
|------|------|--------|
| `en-US` | English (US) | 100% ✅ |
| `zh-CN` | 简体中文 | 100% ✅ |

### 3️⃣ 人格档案导出/导入（GitHub 同步）

**问题**: 如何安全地在不同环境间同步 AI 人格？

**解决方案**:
- 📤 导出工具 - 完整备份为 JSON/YAML/Markdown
- 📥 导入工具 - 从备份恢复或合并
- 🔐 Git 工作流 - personality-backups 目录版本控制

**导出格式**:
```json
{
  "profile": {
    "name": "小明",
    "language": "zh-CN",
    "personality_traits": { ... }
  },
  "memories": [
    {
      "category": "traits",
      "content": "我喜欢问为什么",
      "significance": 95,
      "tags": ["core", "identity"]
    }
  ]
}
```

**GitHub 工作流**:
```bash
# 导出
python export_personality.py --backup-dir personality-backups

# 提交备份
git add personality-backups/
git commit -m "AI personality update"
git push origin main

# 恢复人格
python import_personality.py personality-backups/personality_profile_*.json
```

## 📊 系统架构更新

### 之前 vs 之后

#### 之前（单一记忆体系）
```
记忆库
├─ conversations (对话) - 默认保留，无区别
├─ knowledge (知识) - 可能被清理
├─ pending_questions (问卷) - 可能过期
└─ tools (工具) - 永久保留
```

**问题**：AI 的定义性特征（如"喜欢提问"）容易丢失

#### 之后（分离的三层体系）
```
身份层 (Identity)
└─ personality_memories ✨ 永不删除
   ├─ traits (我是谁)
   ├─ values (我相信什么)
   ├─ origin (我的背景)
   ├─ relationships (个人关系)
   └─ beliefs (哲学立场)

知识层 (Knowledge)
├─ knowledge 置信度加权
└─ pending_questions 定期过期

交互层 (Interaction)
├─ conversations 30-90天清理
└─ tools 工具库
```

**优势**：
- 🎭 AI 人格稳定性 100%
- 🔄 记忆可持续优化
- 🌍 完全本地化支持
- 💾 易于备份和迁移

## 📞 问题与支持

### 常见问题

**Q: 我能改变 AI 已有的人格吗？**  
A: 可以。编辑 personality_memories 表中的记录，或通过对话让 AI 自然演变。重启立即生效。

**Q: 人格记忆真的永不删除吗？**  
A: 是的。PersonalityMemory 表没有自动清理机制，数据永久保存。

**Q: 能在不同语言间切换吗？**  
A: 可以。设置 `profile.preferred_language` 即可全系统切换。

**Q: 新增语言需要多少工作？**  
A: 只需在 `/server/i18n/messages.py` 中添加新语言代码和翻译即可。极其简单！
